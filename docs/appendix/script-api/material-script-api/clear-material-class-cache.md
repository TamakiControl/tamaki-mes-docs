# system.mes.material.clearMaterialClassCache

Clears the [Material Class](../../data-model/material-model/material-class) cache

## Syntax

```python
system.mes.material.clearMaterialClassCache()
```

## Parameters

| Parameter | Type | Nullable | Description                               |
| --------- | ---- | -------- | ----------------------------------------- |
| None      | -    | -        | This method does not take any parameters. |

## Returns

No response is returned.

## Code Examples

```python
system.mes.material.clearMaterialClassCache()
```
