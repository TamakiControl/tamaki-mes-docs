# system.mes.material.clearMaterialCache

Clears the [Material](../../data-model/material-model/material) cache

## Syntax

```python
system.mes.material.clearMaterialCache()
```

## Parameters

| Parameter | Type | Nullable | Description                               |
| --------- | ---- | -------- | ----------------------------------------- |
| None      | -    | -        | This method does not take any parameters. |

## Returns

No response is returned.

## Code Examples

```python
system.mes.material.clearMaterialCache()
```
