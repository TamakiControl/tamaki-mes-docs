# system.mes.location.clearLocationCache

Clears the [Location](../../data-model/location-model/location) cache

## Syntax

```python
system.mes.location.clearLocationCache()
```

## Parameters

| Parameter | Type | Nullable | Description                               |
| --------- | ---- | -------- | ----------------------------------------- |
| None      | -    | -        | This method does not take any parameters. |

## Returns

No response is returned.

## Code Examples

```python
system.mes.location.clearLocationCache()
```
