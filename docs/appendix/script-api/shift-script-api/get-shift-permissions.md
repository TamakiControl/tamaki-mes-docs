# system.mes.shift.getShiftPermissions

## Description
Retrieves a list of all role permission strings related to shifts.

## Returns
| Type         | Description                          |
|--------------|--------------------------------------|
| `List<String>` | List of all shift role permissions.    |

## Code Example
```python
permissions = system.mes.shift.getShiftPermissions()
```
