# system.mes.oee.getOeePermissions

## Description
Retrieves a list of all role permission strings related to OEE.

## Returns
| Type         | Description                          |
|--------------|--------------------------------------|
| List<String> | List of all OEE role permissions.    |

## Code Example
```python
permissions = system.mes.oee.getOeePermissions()
```
