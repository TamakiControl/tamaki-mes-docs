# system.mes.personnel.clearDepartmentCache

Clears the [Personnel Department](../../data-model/personnel-model/personnel-department) cache

## Syntax

```python
system.mes.personnel.clearDepartmentCache()
```

## Parameters

| Parameter | Type | Nullable | Description                               |
| --------- | ---- | -------- | ----------------------------------------- |
| None      | -    | -        | This method does not take any parameters. |

## Returns

No response is returned.

## Code Examples

```python
system.mes.personnel.clearDepartmentCache()
```
