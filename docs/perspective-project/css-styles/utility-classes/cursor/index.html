Cursor

Overview​

These classes control the mouse cursor style when hovering over an element.

Classes​

Class NameDescriptionExample
cursor-autoThe browser determines the cursor to display.
cursor-defaultThe default cursor.
cursor-pointerThe cursor is a pointer.
cursor-waitThe cursor indicates that the program is busy.
cursor-textThe cursor indicates text.
cursor-moveThe cursor indicates that something can be moved.
cursor-not-allowedThe cursor indicates that the requested action will not be executed.
